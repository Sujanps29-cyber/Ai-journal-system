\# AI-Powered Journaling System



\## Overview

This project is an AI-powered journaling system that allows users to write daily journal entries and analyze their emotions using AI techniques.



\## Features

\- Write and save journal entries

\- Analyze emotions from text

\- View previous entries

\- SQLite database storage

\- REST API backend



\## Tech Stack

Frontend: React  

Backend: Node.js + Express  

Database: SQLite  

AI: Basic sentiment analysis service



\## Project Structure



frontend/

\- React user interface



backend/

\- Express API

\- AI emotion service

\- SQLite database
